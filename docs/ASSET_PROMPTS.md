# Asset prompt pack — generating concept-grade pixel art

The enhanced branch ships **original placeholder pixel art** generated by
`scripts/gen_assets.py` (PIL). To reach the *concept-image* look, replace those PNGs in
`public/assets/` with assets from a pixel-art **image model** (Midjourney / SDXL / DALL·E /
Stable Diffusion pixel-art LoRA) or **Aseprite**. Keep the same **file names, sizes, and
transparency**, and the engine renders them with no code change.

> Style anchor for every prompt: *“original pixel art, warm sunset palette, crisp pixels,
> clean readable silhouette, transparent background, no text, game-ready sprite, 1× scale.”*
> All assets must be **original** — do not request Mario/Nintendo/Doodle-Jump likenesses.

## Workflow
1. Generate the images, slice/scale to the sizes above (Aseprite or an online slicer).
2. Drop them into `public/assets/` (overwrite the placeholders).
3. `npm run dev` — the parallax, animation, glow, embers, vignette, and HUD all reuse them.
4. For new animation frame counts, update the `frameWidth`/frame list in
   `src/scenes/PreloadScene.ts` only.
