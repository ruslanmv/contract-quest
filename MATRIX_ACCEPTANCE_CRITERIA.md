# Acceptance criteria (RMD-111)
- `npm ci && npm run build` succeeds; `dist/` is produced.
- `npm run typecheck` passes (no type errors).
- The game loads in the browser with a `<canvas>` and NO console errors.
- The hero falls under gravity, lands on platforms, moves left/right, and jumps.
- HUD shows COINS / SCORE / LIVES and the CONTRACT QUEST title.
- No copyrighted assets. watsonx only for AI coding assistance.
